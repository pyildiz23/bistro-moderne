Bistro Moderne Website
English Version

Description:
A modern restaurant website showcasing the menu, gallery, reservations, and contact information with a fully responsive design.

Features:

Single Page Navigation (SPA): Smooth section switching without page reload.

Multilingual Support: English & French content with language switcher.

Responsive Design: Works seamlessly on desktop and mobile.

Hero Section: Full-screen background with overlay, title, subtitle, and call-to-action.

Menu Section: Starters, Mains, Desserts with dynamic pricing and language.

Gallery Section: Image grid with hover zoom effect.

Reservation Form: Interactive form with success message and reset functionality.

Contact Section:

Colorful icons (address, phone, email, hours)

Google Maps embedded

Responsive layout: Info above, map below on mobile

Navbar & Footer: Responsive navbar with hamburger menu and language switcher; footer with logo and copyright.

Technologies Used: React, React Hooks (useState, useEffect), Lucide React, React Icons, CSS Grid/Flexbox, Google Maps Embed.

Version Française

Description :
Un site web moderne pour restaurant présentant le menu, la galerie, les réservations et les informations de contact avec un design entièrement responsive.

Fonctionnalités :

Navigation SPA : Changement de section sans recharger la page.

Support Multilingue : Contenu en anglais et français avec sélecteur de langue.

Design Responsive : Compatible desktop et mobile.

Section Hero : Fond plein écran avec overlay, titre, sous-titre et bouton d’action.

Section Menu : Entrées, Plats principaux, Desserts avec prix dynamiques et traduction.

Section Galerie : Grille d’images avec effet zoom au survol.

Formulaire de Réservation : Formulaire interactif avec message de succès et réinitialisation automatique.

Section Contact :

Icônes colorées (adresse, téléphone, email, horaires)

Carte Google Maps intégrée

Mise en page responsive : informations au-dessus, carte en dessous sur mobile

Navbar & Footer : Navbar responsive avec menu hamburger et sélecteur de langue ; footer avec logo et copyright.

Technologies Utilisées : React, React Hooks (useState, useEffect), Lucide React, React Icons, CSS Grid/Flexbox, Google Maps Embed.
