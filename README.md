# Bistro Moderne Website

A modern restaurant website showcasing the menu, gallery, reservations, and contact information with a fully responsive design.

## 🌐 English Version

### Features
- **Single Page Navigation (SPA)** – Smooth section switching without page reload  
- **Multilingual Support** – English & French content with language switcher  
- **Responsive Design** – Works seamlessly on desktop and mobile  
- **Hero Section** – Full-screen background with overlay, title, subtitle, and call-to-action  
- **Menu Section** – Starters, Mains, Desserts with dynamic pricing and language  
- **Gallery Section** – Image grid with hover zoom effect  
- **Reservation Form** – Interactive form with success message and reset functionality  
- **Contact Section** – Colorful icons (address, phone, email, hours), Google Maps embedded, responsive layout  
- **Navbar & Footer** – Responsive navbar with hamburger menu and language switcher; footer with logo and copyright

### 🛠 Technologies Used
- **React** & **React Hooks** (`useState`, `useEffect`)  
- **Lucide React** & **React Icons**  
- **CSS Grid & Flexbox**  
- **Google Maps Embed**
